========================================
HOW TO CREATE YOUR FIRST ADMIN USER
========================================

You need to change your role to ADMIN in the database.

========================================
OPTION 1: Using Render Shell (EASY)
========================================

1. Go to Render Dashboard
2. Click your service: lovewaving
3. Click "Shell" tab
4. Type this command:

   npx prisma db execute --stdin

5. Paste this SQL:

   UPDATE "User" SET role = 'ADMIN' WHERE email = 'your-email@example.com';

6. Replace 'your-email@example.com' with YOUR actual email
7. Press Enter
8. Done!
