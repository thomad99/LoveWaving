WHAT WAS ADDED:
========================================

1. QR Code Component
   - Shows on admin event detail page
   - Generates QR code for public join URL
   - Copy link button
   - Download QR code button

2. Public Join Page
   - URL: /join/[eventId]
   - Shows event details
   - Shows waiver preview
   - Button to complete waiver

========================================
HOW IT WORKS:
========================================

ADMIN VIEW:
1. Go to event detail page
2. See QR code card
3. Click "Copy Link" or "Download QR"
4. Share with participants

USER FLOW:
1. Scan QR code or click link
2. See event details on /join/[id]
3. Click "Complete Waiver to Join"
4. Redirected to sign page
5. Signs waiver
