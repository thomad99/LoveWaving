# LoveWaving - Render Service Configuration

## Service Type
Web Service

## Configuration Details

### Environment
Language: Node (NOT Go!)
Branch: main
Region: Oregon (US West) - or your preferred region
Root Directory: (leave empty)

### Build Command
npm install && npx prisma generate && npm run build

### Start Command
npm start

### Instance Type
Free (to start with, can upgrade later)

### Auto-Deploy
✅ Enabled
✅ Deploy from branch: main

## Important Notes

- DO NOT select "Go" as the language
- Select "Node" for Next.js applications
- The build command installs npm packages, generates Prisma Client, and builds Next.js
- Start command runs the Next.js production server
- Leave Root Directory empty unless using a monorepo

## After Service Creation

Don't forget to add environment variables:
- DATABASE_URL
- NEXTAUTH_URL
- NEXTAUTH_SECRET
- AWS credentials (if using S3)
- Google OAuth (if using)
